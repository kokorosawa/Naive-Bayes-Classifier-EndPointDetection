# 語音切割方法說明 🎯

## 資料預處理 📊
1. 使用 Gaussian Naive Bayes 作為基礎模型
2. 對音訊資料進行分割，frame size 設定為 1024 samples
3. 使用 192 重疊進行特徵提取
4. 對特徵進行標準化處理

## 模型訓練 🤖
1. 將資料集以 8:2 比例分割為訓練集和測試集
   - 調整容許誤差範圍至 1000 samples == 0.0625ms
2. 嘗試過 mfcc , spectrum, 效果比不過單純分割成音框

## 評估方法 📈
1. 計算起始點和結束點的預測準確率
2. 使用時間容許誤差判定 (±1000 samples == 0.0625)
3. 計算整體辨識率
